# Portfolio React Pro - Développeuse Full Stack


## 🎯 Aperçu

Portfolio professionnel moderne développé avec React et TypeScript, présentant les compétences, projets, expériences et certifications d'une développeuse Full Stack spécialisée en applications web, mobile et Big Data.

**Design :** Premium Minimalist - Palette zinc/bleu, sans mauve/violet  
**Performance :** Optimisé Vite + TypeScript  
**Responsive :** Mobile-first avec Tailwind CSS  

## 🛠 Stack Technique

| Technologie | Version | Utilisation |
|-------------|---------|-------------|
| React | 18.3 | Bibliothèque UI |
| TypeScript | 5.5 | Typage statique |
| Vite | 5.4 | Build tool |
| Tailwind CSS | 3.4 | Styling |
| React Router DOM | 6.26 | Navigation |
| Lucide React | 0.468 | Icônes |
| ESLint | 9.9 | Linting |
| Prettier | 3.3 | Formatage |

## ✨ Fonctionnalités

### 📱 Pages Principales
- **Accueil** - Présentation avec photo de profil
- **Expérience** - Stages enseignement et entreprises
- **Projets** - 9 projets avec filtrage par catégorie
- **Éducation** - Timeline formations académiques
- **Certifications** - 8 certifications avec images
- **Contact** - Formulaire et informations

### 🎨 Design Features
- Layout asymétrique moderne
- Cards avec effets hover
- Timeline chronologique
- Filtres dynamiques
- Backdrop blur sur navigation
- Animations fluides

### 📊 Données Centralisées
- `src/data/experience.ts` - Expériences professionnelles
- `src/data/projects.ts` - Projets réalisés
- `src/data/education.ts` - Formations académiques
- `src/data/certifications.ts` - Certifications
- `src/data/profile.ts` - Informations personnelles

## 🚀 Installation

### Prérequis
- Node.js ≥ 18.0.0
- npm ≥ 9.0.0 ou yarn ≥ 1.22.0
- Git

### Étapes d'installation

```bash
# 1. Cloner le dépôt
git clone https://github.com/votre-username/mon-portfolio.git
cd mon-portfolio
```

# 2. Installer les dépendances
```
npm install
```
# 3. Lancer le serveur de développement
```
npm run dev
```
# 4. Ouvrir dans le navigateur
```
http://localhost:5173
```
